# 🧠 EmpathAI – Emotion-Aware Customer Care Agent

> **AI-powered customer support that understands emotions before responding.**

EmpathAI is an intelligent customer care platform that uses Artificial Intelligence and Natural Language Processing (NLP) to detect customer emotions in real time and automatically adapt its responses. Unlike traditional chatbots, EmpathAI provides empathetic, context-aware, and emotionally intelligent conversations while helping businesses improve customer satisfaction and reduce support workload.

---

## 🚀 Project Overview

Customer support bots often fail to understand customer emotions, leading to poor user experiences and unnecessary escalations.

EmpathAI addresses this challenge by:
- Detecting customer emotions
- Predicting escalation risks
- Adapting AI response tone dynamically
- Providing real-time analytics
- Assisting support teams with AI-generated insights

---

## 🎯 Problem Statement

Traditional customer support systems respond with generic answers regardless of customer emotions. This can frustrate users, increase support costs, and reduce customer satisfaction.

Businesses need an intelligent support solution capable of understanding customer emotions and providing human-like, empathetic responses.

---

## 💡 Solution

EmpathAI combines Generative AI, Sentiment Analysis, and Intelligent Automation to create emotionally aware conversations.

The system analyzes every customer message, detects emotions such as anger, frustration, urgency, happiness, and confusion, then dynamically adjusts the chatbot's communication style.

If customer frustration increases, the platform predicts escalation and recommends human intervention before the conversation becomes critical.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/empathai.git

cd empathai
```

## Install Frontend

```bash
cd frontend

npm install
```

## Install Backend

```bash
cd backend

npm install
```

---

## Environment Variables

Create a `.env` file.

```
PORT=5000

MONGO_URI=your_mongodb_connection

JWT_SECRET=your_secret

GEMINI_API_KEY=your_api_key

CLIENT_URL=http://localhost:3000
```

---

## Start Backend

```bash
npm run dev
```

---

## Start Frontend

```bash
npm run dev
```
